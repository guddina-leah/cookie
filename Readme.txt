branch is take your own version of the file to change
commit is save your changes to the project
push puts file in the online repo
merge merges your changes with branch or master

git help <- lists common commands

New Project:
mkdir ~/projname
cd ~/projname
git init <- tell git about this dir
touch file.ext <- creates an empty file
git status <- reports info
git add file.ext <- adds file to what git watches
git commit file.ext <- commit to the local dir
git remote add origin https://github.com/uname/projname.git <-connect dir to repo
git push -v origin master <- push changes to online repo
